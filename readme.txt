== Description ==
This plugin allows you to integrate your WhatsApp experience directly into your website. This is one of the best ways to connect and interact with your customer, you can offer support directly as well as build trust and increase customer loyalty.

[View Demo Here](https://demo.essentialplugin.com/essential-chat-support/).

WhatsApp connectivity for WordPress websites! Engage customers and generate leads!

= Available Features : =
* Display Mode `Single Mode` OR `Multi Mode`.
* Display Type `Agent`, `Group` OR `Both`.
* Display chat on global location
* Two chat toggle position (Right Bottom, Left Bottom)
* Display On(Desktop, Mobile, Both)
* Contact shortcode
* Display contact in grid view
* Customize Designs
* Custom CSS box
* Fully responsive
* 100% Multilanguage
* Lightweight, Fast & Powerful
* Work with any WordPress Theme

== Installation ==

1. Upload the 'essential-chat-support' folder to the '/wp-content/plugins/' directory.
2. Activate the "Essential Chat Support" plugin through the 'Plugins' menu in WordPress.
3. Create your chat box.
4. Select "Chat Box" from Plugin Settings > Display Rules
